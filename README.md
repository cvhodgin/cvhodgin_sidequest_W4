## Side Quest Week 4: Mountain Maze 
---
## Description

> This project lets players move through a maze using keyboard input (arrow keys).  
> It provides verbal feedback to communicate what is happening when the user interacts with a simple visual object, such as a circle (representing a crystal) or a red triangle (representing a mountain). 
> It explores the gameplay of Celeste through the use of arrow movement, interaction with game objects, and the addition of a second level that loads
automatically when the first is finished. 

---
## Interaction Instructions

> The player starts by pressing arrow keys to move in respective direction (ie. when pressed, arrow key pointing down moves player down). 
> The player can watch out for visual verbal feedback in the caption at the bottom of the screen to understand what the simple objects in the game represent. 
> To make it to the next level the player must land on the 'Exit' tile of the maze. 
> Along the way, the player has the option of landing over the blue circles to gain points and increase their score. 
> `R` to reset the canvas at the end of the two levels.
